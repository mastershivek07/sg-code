# Changelog

All notable changes to SG CODE will be documented in this file.

## [1.0.0] - 2026-04-15

### Added
- Initial release
- Multi-provider support: ChatGPT OAuth and Google AI (Antigravity)
- Native terminal with xterm.js (256-color, web links)
- Multi-tab session management
- Model switching with `/model` command
- 15+ AI models: GPT, Claude, Gemini families
- Auto-updater for seamless updates
- macOS (DMG) and Windows (NSIS installer) builds
- Green hacker theme with SG CODE branding
- User analytics via Supabase (anonymous, for product improvement)
